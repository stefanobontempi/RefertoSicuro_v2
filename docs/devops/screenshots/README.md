# Branch Protection Screenshots

Questa directory contiene screenshot della configurazione GitHub Branch Protection.

## File da includere

1. `main-branch-protection.png` - Configurazione completa main branch
2. `develop-branch-protection.png` - Configurazione completa develop branch
3. `pr-approval-required.png` - Esempio PR bloccata senza approval
4. `status-checks-required.png` - Esempio status check requirement

## Come prendere screenshot

1. Vai su GitHub → Settings → Branches
2. Click su Edit per la rule `main`
3. Scroll e cattura l'intera configurazione
4. Salva come `main-branch-protection.png`
5. Ripeti per `develop`
